Woogles Embed Widget Demo

About the Embed Widget

The Woogles embed widget allows you to embed a complete Scrabble game viewer in your blog, website, or application without any external dependencies or API calls.

Features:

  • Self-contained - no XHR requests needed
  • Interactive game replay with prev/next controls
  • Responsive design
  • Light and dark theme support
  • Customizable display options

Basic Embed Example

This example shows a basic embed with default settings:

<div id="woogles-embed-game1"></div> <script> window.WooglesEmbed = { gameId: "example-game-1", containerId: "woogles-embed-game1", gameDocument: /* game data would be here */, options: { width: 600, showControls: true, showScores: true } }; </script> <script src="/dist/embed.js"></script>

Customized Embed Example

This example shows an embed with custom options:

<div id="woogles-embed-game2"></div> <script> window.WooglesEmbed = { gameId: "example-game-2", containerId: "woogles-embed-game2", gameDocument: /* game data would be here */, options: { width: 800, height: 600, showControls: true, showScores: true, showMoveList: true, theme: 'dark' } }; </script> <script src="/dist/embed.js"></script>

How to Get the Embed Code

To embed a game from Woogles.io:

  1. Navigate to the game you want to embed
  2. Click the "Share" or "Embed" button
  3. Copy the generated embed code
  4. Paste it into your HTML page

Server Endpoint: GET /embed/generate/{gameId}

This endpoint returns the complete HTML snippet with the game data embedded.

Embed Options

Option Type Default Description
width number 600 Width of the embed in pixels
height number 700 Height of the embed in pixels
showControls boolean true Show navigation controls
showScores boolean true Show player scores
showMoveList boolean false Show current move details
theme string 'light' Color theme ('light' or 'dark')
